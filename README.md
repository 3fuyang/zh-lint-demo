# Zhlint Demo

An intuitive playground for [zhlint](https://jinjiang.dev/zhlint/).

The diffing is provided by [diff](https://github.com/kpdecker/jsdiff).

## How to Run

```bash
corepack enable

pnpm i

pnpm dev

```
